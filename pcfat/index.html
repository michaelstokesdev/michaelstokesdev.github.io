Requirements to run

There are a few requirements for this tool. They are:

  • Python 3
  • The MatPlotLib library

How to use the tool

This tool is very simple and intuative to use. It can be operated either through the command line or through the built in GUI. There are a number of different options for how to use the tool, including using it as a library in your own programs by importing it.

GUI Operation

Operating this tool with the GUI should be simple and intuative for most people. To do this you simply run the program with no command line arguments and the GUI will start automatically. Currently through the GUI the only option is to generate a graph of the character frequencies.

Command Line Operation

Command line operation of this tool should be fairly simple. To run using the command line you simply type the python command, followed by the input file, followed by any desired arguments. For example:

python3 test.txt -g

There are a number of different arguments that can be used. Leaving the arguments blank will open the default GUI environment. The arguments for the program are as follows:

  • -h - Provides a list of all the command line arguments that can be used and the parameters that they require.
  • -o [filename] - This will output all of the percentage values in a JSON dictionary format to a specified file. This will not overwrite existing files.
  • -g - This will display a graph of the specified input file using the MatPlotLib library.

Use as a library

This tool can also be easily used as a library to perform frequency analysis on any string. To do this simply take the pcfat.py file, place it into a directory in your project and then import the file and use it's functions. Some potentially useful functions are:

  • getTotals(text) - This returns a dictionary of all the characters in the given text and a count of the total number of non whitespace characters in the text.
  • getPercentage(dictionary from getTotals, total character count) - This returns a dictionary containing percentage values for how much of the text is made up of each character.
  • getFrequencies(text) - This returns the result of putting the return value of getTotals into getPercentage with all whitespace characters removed. i.e. text > whitespace removed > getTotals > getPercentage > return.
